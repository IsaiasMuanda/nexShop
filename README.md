# NexShop

SPA de e-commerce construída com React + Vite, com catálogo de produtos, busca em tempo real e carrinho funcional com drawer lateral.

## Tecnologias

- React 18
- React Router DOM v7
- Vite
- CSS customizado com variáveis (design system próprio)

## Funcionalidades

- Listagem e busca de produtos em tempo real
- Página de detalhe por produto
- Carrinho com drawer lateral, controle de quantidade e subtotal
- Navegação por rotas (Home, Produtos, Posts, Sobre)
- Consumo de API externa (JSONPlaceholder) na página de Posts
- Layout responsivo

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
